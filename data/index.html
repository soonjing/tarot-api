Rider-Waite-Smith Cards API (Modified by Soon Jing)

Using data from The Pictorial Key to the Tarot by Arthur Edward Waite.

View YAML documentation.

Examples:

Get all cards: https://soonjing.herokuapp.com/api/v1/cards

Search all cards with "difficult" in reversed or upright meanings: https://soonjing.herokuapp.com/api/v1/cards/search?meaning=difficult
